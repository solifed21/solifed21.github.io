1편에서 디스코드 봇을 만들고 토큰을 준비하셨나요? 2편에서는 그 토큰을 OpenClaw에 심고, 실제로 AI 에이전트를 디스코드에서 깨워보는 과정을 진행하겠습니다. ㅋㅋ

---

## 1단계: 봇 토큰 설정하기

OpenClaw가 내 봇을 제어할 수 있도록 토큰을 알려줘야 합니다. 두 가지 방법이 있는데, 편리한 방법을 선택하세요.

### 방법 1: 환경 변수 사용 (서버 운영 시 권장)
터미널 설정 파일(`.zshrc` 또는 `.bashrc`)에 토큰을 등록합니다.

```bash
export DISCORD_BOT_TOKEN="여기에_복사한_토큰_붙여넣기"
```

### 방법 2: 설정 파일(`config.yaml`) 수정
OpenClaw 설정 파일(`~/.openclaw/config.yaml`)을 직접 수정합니다.

```yaml
channels:
  discord:
    enabled: true
    token: "여기에_복사한_토큰_붙여넣기"
```

---

## 2단계: 디스코드 서비스 가동

이제 준비가 끝났습니다. OpenClaw의 디스코드 채널 서비스를 시작해 봅시다!

터미널에서 아래 명령어를 입력하세요:

```bash
openclaw discord start
```

성공적으로 연결되면 터미널에 다음과 같은 메시지가 뜹니다:
`🤖 Discord channel started: My-OpenClaw-Bot (ID: ...)`

만약 에러가 난다면 1편의 **Message Content Intent** 설정을 다시 확인해 보세요!

---

## 3단계: AI와 첫 대화 나누기 (DM 및 채널)

### 1. 개인 DM으로 대화하기
디스코드에서 내 봇을 찾아 말을 걸어보세요. "안녕?"이라고 보내면 봇이 답장을 할 거예요.

**참고:** 기본 설정상 처음 DM을 보내면 **'Pairing Code'**를 요구할 수 있습니다. 보안을 위해 터미널에서 `openclaw pairing approve discord`를 실행하여 승인해 주시면 됩니다.

### 2. 서버 채널에서 대화하기
봇이 초대된 서버의 채널에서 봇을 **@멘션**하고 말을 걸어보세요.

예: `@My-OpenClaw-Bot 오늘 서울 날씨 어때?`

---

## 4단계: 편리한 슬래시(/) 명령어 활용

OpenClaw 디스코드 봇은 기본적으로 여러 슬래시 명령어를 지원합니다. 채팅창에 `/`를 입력해 보세요.

- `/status`: 현재 에이전트의 상태와 비용 확인
- `/config`: 실시간 설정 변경
- `/reset`: 대화 문맥 초기화
