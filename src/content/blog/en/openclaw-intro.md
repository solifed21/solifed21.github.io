## What is OpenClaw?

OpenClaw is an open-source framework for AI agents. Simply put, it lets AI actually do work on your computer.

## Why do you need it?

Traditional AI chatbots could only chat. With OpenClaw:

- Read and write files
- Execute terminal commands
- Analyze and modify code
- Search the web

## Features

1. **Open Source**: Free for everyone
2. **Extensible**: Add features via plugins
3. **Secure**: Runs locally, no data leaks

## Use Cases

- Automate development tasks
- Write and edit documents
- Code review
- Handle repetitive tasks

The AI agent era is here. Start with OpenClaw.
