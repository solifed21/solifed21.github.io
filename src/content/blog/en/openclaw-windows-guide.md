## Requirements

- Windows 10 or 11

## Installation

### Step 1: Open PowerShell

Search "PowerShell" in Start menu. Click "Run as administrator".

### Step 2: Check winget

Windows 11 has it pre-installed. For Windows 10, update "App Installer" from Microsoft Store.

### Step 3: Install OpenClaw

In PowerShell:

```powershell
winget install openclaw
```

Done.

## Running OpenClaw

In PowerShell or Command Prompt:

```powershell
openclaw
```

That's it.

## Troubleshooting

- Close and reopen PowerShell
- Restart your computer
- Make sure winget is properly installed

Not hard at all. Just follow along.
